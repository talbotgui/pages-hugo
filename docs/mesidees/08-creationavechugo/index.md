## La structure

Le plus simple est toujours de partir d'une base de travail simple.
Pour cela, sont disponibles plusieurs base :
* [SIMPLE] (https://github.com/talbotgui/creation-hugo/tree/simple) : un site Hugo de 2 pages avec le thème learn
* [INDUS] (https://github.com/talbotgui/creation-hugo/tree/indus) : le site SIMPLE mais industrialisé avec NPM pour démarrer le site sur Windows, Unix ou simplement générer le site
* [SEARCH] (https://github.com/talbotgui/creation-hugo/tree/search) : le site INDUS avec la fonction de recherche basée sur LUNR

Un exemple d'usage réel est disponible [ici] (https://github.com/talbotgui/pages-hugo/tree/master).

## Les astuces à connaître

basiques :
 nouvelle page = redémarrage
 ...

propres au thème :
 chapitre:true => texte centré
 next et prev nécessite une bonne organisation
 outil de check des liens
 ...
